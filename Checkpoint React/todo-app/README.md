# Todo App

An intermediate Todo app built with React functional components and hooks.

## Features

- Add new todos
- Mark todos as completed or pending
- Delete todos
- Filter todos by status (All, Pending, Done)
- Display remaining todos count

## Components

- **App**: Main component managing state and handlers
- **TodoForm**: Form for adding new todos with controlled input
- **TodoList**: Renders the list of todos
- **TodoItem**: Individual todo item with toggle and delete functionality

## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npm start`
3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build

To create a production build: `npm run build`

## Technologies Used

- React 19
- React Hooks (useState)
- CSS for styling
